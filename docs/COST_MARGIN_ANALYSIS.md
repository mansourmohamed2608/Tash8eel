## 2. تكاليف الرسائل (Meta Cloud API)

| نوع الرسالة                                          | التكلفة         | ملاحظات                         |
| ---------------------------------------------------- | --------------- | ------------------------------- |
| **رسائل الخدمة** (رد على العميل خلال 24 ساعة)        | **مجاني**       | 0 ج.م — الأساس                  |
| **رسائل الأدوات** (OTP, تحديث طلب) داخل نافذة الخدمة | **مجاني**       | مجاني منذ أبريل 2025            |
| **رسائل تسويقية** (Template)                         | ~3.70 ج.م/رسالة | تُحسب بشكل منفصل                |
| **رقم WhatsApp**                                     | **مجاني**       | Meta Cloud API — لا تكلفة شهرية |

**🔑 الخلاصة:** 95%+ من الرسائل (محادثات الطلبات) = **مجانية**. فقط رسائل التسويق الاستباقية تكلف.
